manyfold-dark-ui
A dark, minimal frontend UI for Manyfold self-hosted STL library servers.
![screenshot](screenshot.png)
Features

Dark theme with yellow accent
Grid and list view
Search and sort
Paginated model browsing
Connects via Manyfold OAuth2 API
No dependencies — pure HTML/CSS/JS
Credentials stored in localStorage only

Usage

Download index.html
Open in any browser
Enter your Manyfold server URL and API credentials
Click Connect

Setup in Manyfold

Go to Settings → API Keys → New API Key
Set scope to read and public
Copy the Client ID and Client Secret into the app
